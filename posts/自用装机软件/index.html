自用装机软件

Jun 25 2021-06-25T10:37:00+08:00 by NagaseYami
Updated Aug 9 2021-08-09T01:13:41+08:00 9 min

自从买了笔记本电脑之后发现老台式上有很多常用的便利软件都要装到新电脑上，做一个记录，也顺便分享一下。
都是Windows用的，部分也有Mac版，我就不专门查了。
除了部分收费软件的免费版可能会有一点点广告（GUI上或者右小角弹窗），基本上都尽量挑的免费无广告开源。

ocenaudio

用的比较少，一般是用作Youtube上扒下来的音乐做点简单的剪辑用的。
功能其实还挺全的，毕竟我不是专业混音或者音频处理。

balenaEtcher

给树莓派的SD卡刷镜像用的，这个就没啥说的了。
最早是用的树莓派官方的刷镜像工具，偶然有一次刷第三方OpenWrt固件的时候总失败，就换了这个。

Bitwarden

免费密码管理软件。
最早我用的是LastPass，后来因为LastPass限制免费用户只能在桌面端和移动端二选一使用，就只好转移阵地了。
除了嗅探相比LastPass差了不少（经常在我登陆网站之后不跳出保存该网站密码的提示）其他该有的都有，而且也支持导入LastPass的文件，整个转移过程还算轻松。
当然也支持桌面端和移动端，桌面端支持Windows Hello，移动端支持指纹和人脸识别。

Calibre

用处只有一个，就是用来破解Kindle电子书的，需要老版本来装一个特定的插件，所以我装的是Portable。
以后专门开一个文章来记录一下用法好了。

CCleaner

常见的垃圾清理基本上都能用这个完成。

Cheat Engine

最牛逼的游戏作弊器没有之一。
偶尔打一些疯狂刷刷刷的游戏的时候会用这个来节省一下我并不值钱的时间。

Clash for Windows

懂得都懂。

DAEMON Tools Lite

有一说一，就是个黄油安装器。

Everything

因为我自己电脑文件整理的还不错所以并不是用来搜一些文件的存放位置。
而是用于清理一些软件卸载后的残留文件。
基本上用一些正则表达式就能搜到想要的东西。

Fork

自认为最好用的可视化Git。

ImageGlass

免费开源的看图软件，主要好在一个小巧精致不花里胡哨。
毕竟我就看个图，要那么多有的没的功能也没啥用。
看漫画也比较方便。

Neat Download Manager

虽然我老电脑上是买了IDM的，但是一个是感觉IDM小贵，二是实际上IDM的大部分高级功能我也用不上，就有点亏。
NDM基本上能完美代替IDM，而且也是个小巧精致的软件，不到1M。
缺点是如果你的浏览器是Chrome则不能使用Youtube视频嗅探功能（毕竟Chrome插件商店是不允许任何Youtube视频下载插件上架的）。

Kindle

对没错我就是正版电子书的受害者。
花正版的钱买最垃圾的低分辨率JPG漫画。

Notion

世界上最好的笔记类效率工具没有之一。

VirtualBox

虽然现在虚拟机也有很多其他的方案了比如HyperV啥的，不过因为电脑自带的是家庭版Win10也懒得再刷个旗舰版，就VBox继续用了。
而且相较于没怎么用过的HyperV来说我还是更习惯VBox。

PotPlayer

视频播放器。
需要配合XySubFilter和madVR使用来达到最佳的看动画片体验。
我真应该再VCB-S挂掉之前把他们的PotPlayer设置教程转载过来存一份的。
说没就没，真的可惜。

qBittorrent

BT下载。
现在BT生态确实不太行，以后硬盘价格下来了组个NAS挂BT玩。 未来也想玩玩PT。

ScreenToGif

开源免费。 录频然后转成Gif。偶尔有用，但是也没太大用。
毕竟超过6M的Gif图片想发QQ就只能传文件了。
只能想办法压低质量，但是那样又看不清了。

TeamViewer

最好用的远程控制电脑的软件。
虽然我用Steam Link手机远程电脑打黄油用的更多。

Winaero Tweaker

其实就是一个Windows控制面板的增强版。 可以用来修改很多控制面板里没有的系统设置，具体的实现方式也能在软件官网查到。

WinSCP

SFTP/FTP文件传输软件。
一般是用于一些FTP下载，或者是连自己的服务器传输文件。

Visual Studio Code

2021年了还搁这儿用Windows自带的记事本呢？

FileConverter

Github 这个名字就叫FileConverter，为了防止搜不到姑且贴一个Github连接。
简单好用的格式转换软件，常用的格式都能转，安装之后，支持的文件的右键菜单里就会有转格式的选项了。 具体转格式的配置也能调一些，不过比较简易，想要专业一点的那还是建议其他收费软件比较稳妥。
一般我就是做手机铃声之类的时候用。

Snipaste

国人做的一个免费（商用收费）截图贴图软件。
建议在Microsoft Store里下载，这样就有自动更新了。
打游戏的时候还挺常用的，把攻略截个图然后贴出来置顶，就能全屏玩游戏的时候看了（

ReNamer

比百度出来的那些批量重命名软件要高级的多。
支持很多规则，我一般用于漫画的图片文件的批量重命名。

GIMP

开源免费图片处理/绘画软件。 说实话我实在是懒得再去弄D版PS了。
作为一个不画画也不玩修图的人，滤镜画笔插件我都用不上，实在是没必要硬磕PS。
常见图片处理功能该有的都有，也能顺便帮我做点图片格式转换比如转pdf。
